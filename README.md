# bitscube_intellectual

## 需求
 1. 首页：搜索、项目列表
 2. 搜索：搜索结果、分页加载
 3. 项目列表：当前任务流、人才详情页、部门详情页、项目详情页
 4. 智库：人才、项目
 5. 我的：基本信息、个人标签、参与项目、设置等
